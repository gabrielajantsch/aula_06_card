# ⚔️ Arkon, o Guerreiro - Card TCG

Este projeto foi desenvolvido como parte de um desafio de **CSS** e **Git** para a faculdade de Engenharia de Software. O objetivo foi realizar um "Glow Up" em um card de personagem, transformando um layout básico em uma interface temática inspirada em jogos de cartas colecionáveis (TCG).

## 🚀 Tecnologias Utilizadas
* **HTML5**: Estrutura semântica (mantida original conforme as regras).
* **CSS3**: Estilização, Flexbox, Box Model e efeitos visuais (Aura/Glow).
* **Git & GitHub**: Controle de versão e histórico de evolução.

## 🎨 O que foi implementado?
* **Identidade Visual**: Cores terrosas e bordas douradas.
* **Tipografia**: Uso de fontes com serifa para um ar medieval.
* **Layout Responsivo**: Ajustes de `padding` e `gap` para garantir o "respiro" do texto.
* **Efeitos**: Sombras compostas (`box-shadow`) para criar o brilho de item lendário.

## 🛠️ Como visualizar
1. Clone o repositório.
2. Abra o arquivo `index.html` em qualquer navegador.
